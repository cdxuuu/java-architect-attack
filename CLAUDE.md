## 仓库概述

这是一个 Java 架构师面试知识库，包含分布式系统设计核心主题的详细内容。仓库包含以下主题的面试准备材料：
- CAP 定理
- 分布式事务
- 分布式锁
- 缓存架构

仓库包含以下 Markdown 文件：
- `架构师面试-Day01-CAP定理.md` - CAP 定理详解
- `架构师面试-Day02-分布式事务.md` - 分布式事务理论与实践
- `架构师面试-Day02-分布式事务-梳理.md` - 分布式事务知识梳理

- `架构师面试-Day04-缓存架构.md` - 缓存架构设计
- `架构师面试-Day04-缓存架构-梳理.md` - 缓存架构知识梳理

## 架构师面试每日练习
  训练节奏：5天新题 + 1天串联 + 1天深挖
  每周新建一个文件夹：以xx年xx月xx周开始，后续的计划都根据当前日期进入到具体的目录下进行
  格式：周一到周五每日文档分为两个，分别是问答和架构师角度梳理，周六叫场景串联，周日叫架构深挖，文档名字结尾带上日期
  已完成：查看当前年和周对应的文件夹里面的记录
  今天是周六：串联日，已完成本周4个主题的串联整合
  周一到周五：架构出题日和梳理日（要排除之前出过的题目）
  周六：串联日，将本周的新题和场景结合做串联，举出几个场景共我选择
  周日：以架构视角深挖本周的内容，训练我的架构师习惯
