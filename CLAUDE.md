## 仓库概述

这是一个 Java 架构师面试每日练习知识库，采用"5天新题 + 1天串联 + 1天深挖"的训练节奏，按周组织内容。每周围绕分布式系统核心主题出题，用户作答后整理为问答和架构师梳理两个文档，并分析能力差距与补足方向。

已覆盖主题：
- **2026年05月第3周**：CAP 定理、分布式事务、分布式锁、缓存架构 | 串联整合 | 架构深挖-Raft协议
- **2026年05月第4周**：消息队列 | 进行中...

## 文件结构

按周组织，每周一个目录（如 `2026年05月第3周/`、`2026年05月第4周/`），目录内包含：
- `架构师面试-DayXX-主题.md` - 问答与练习整理
- `架构师面试-DayXX-主题-梳理.md` - 架构师角度梳理
- `架构师面试-Day06-串联整合.md` - 周六场景串联
- `架构师面试-Day07-架构深挖-XX.md` - 周日深挖某个点
- `架构师面试-能力差距梳理.md` - 当周能力差距分析

## 架构师面试每日练习

**训练节奏**：5天新题 + 1天串联 + 1天深挖，每周新建文件夹（如 `2026年05月第4周/`）

**每日安排**：
- 周一到周五：出题日，每日两个文档（问答 + 架构师梳理），排除已出过的题目
- 周六：串联日，将本周新题和场景结合做串联，举出几个场景供选择
- 周日：深挖日，以架构视角深挖本周内容，训练架构师思维

**出题优先级**（排除已覆盖主题后，按以下顺序选题）：
1. 用户业务直接相关的主题（秒杀、支付、服务注册发现等）
2. 面试超高频考点（MQ选型、微服务、限流降级等）
3. 前序主题的延伸（如CAP→Raft，分布式事务→消息最终一致性）
4. 架构师进阶主题（系统设计、容量规划、故障排查等）

**练习要求**：每道题作答后需指出与架构师水平的差距及补足方向

**用户业务背景**：蛋壳钱包（TCC分布式事务）、平安健康险保险商城、IM在线客服、医疗在线问诊、国美e卡资金路由、啄木鸟云健康智慧公卫体检、内部管理系统（SETNX裸实现）、ToC核心业务（Redisson）、完整秒杀系统经验
